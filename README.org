* Конспекты
Этот репозиторий служит для хранения конспектов. Конспекты могут находиться в двух папках:
+ В =school/= (конспекты для школы)
+ В =guides/= (конспекты по разным внешкольным темам)

* Зеркала
По сути, зеркало - это тот же репозиторий, но на другой платформе. Для меня второй репозиторий нужен по техническим причинам - GitLab плохо отображает документы с расширением =.org=, в которых и написаны конспекты. В GitHub'е же нет некоторых полезных фишек, но он лучше отображает документы (работают ссылки по ID, есть подсветка синтаксиса). Поэтому GitHub лучше использовать для чтения файлов, а GitLab - для поддержания (меинтейна) проекта. Вот ссылки:

[[https://gitlab.com/unenSCRIBE/synopses/][Оригинальный репозиторий]] на GitLab
[[https://github.com/undercoverOrg/synopses/][Репозиторий-зеркало]] на GitHub
